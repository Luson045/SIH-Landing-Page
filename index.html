GuardianGo

Smart Tourist Safety Monitoring & Incident Response System using AI, Geo-Fencing, and Blockchain-based Digital ID. Built for safe and secure tourism.

Watch Demo

Key Features

Blockchain-secured IDs

Digital Tourist IDs issued at entry points to ensure tamper-proof identity and travel data.

Geo-fencing & AI Alerts

Real-time monitoring with anomaly detection and proactive alerts in high-risk zones.

Panic Button

Instant live location sharing with police and emergency contacts for rapid response.

Automated e-FIR

Seamless emergency report filing directly from the system to concerned authorities.

Offline Fallback

SMS-based emergency alerts in remote/no-internet regions for uninterrupted safety.

Multi-Lingual Support

Accessible for diverse tourists including elderly and differently abled travelers.

Technical Approach

Languages

Python (backend, data), Dart (Flutter app), Solidity (smart contracts).

Frameworks

TensorFlow, Flutter, Firebase, FastAPI, Google Maps & fl_chart.

Blockchain

Polygon-based smart contracts for immutable and trusted travel data.

Location Services

Fused Location Provider for efficient real-time geo-tracking.
